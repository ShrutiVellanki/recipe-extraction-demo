## ⚙️ The Prototype

- **Python app** that extracts recipe data from PDF to JSON
- Powered by **GPT-5**, **Python**, **PyMuPDF**

Let's run `extract.py`!

| Step               | Description                                                                              |
| ------------------ | ---------------------------------------------------------------------------------------- |
| **1. Extract**     | PDFs + PyMuPDF  ➡️ Raw Text                                          |
| **2. Parse + Structure**       | 🤖 AI + Prompts + Schema  ➡️  Structure                                 |
| **3. Validate**    | Check the generated output against the `schema.json`; non-conforming outputs are rejected        |

🔑 **Consistency Is Key**
